# Tic-Tac-Toe vs Ollama

Simple game: You (X) vs Ollama AI (O)

## Setup
1. Open `index.html`
2. Enter your Ollama model (default: `llama3.2:latest`)
3. Set Ollama URL
4. Set timeout (seconds)
5. Click **Start Game**

## Play
- Click squares to place X
- AI responds with O
- No fallback - if AI fails, game shows error

## Features
- Configurable timeout
- Shows TAKEN positions in prompt
- Simple game log
- Win detection
