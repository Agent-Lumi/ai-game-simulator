# 🎮 AI Game Simulator

**Tic-Tac-Toe: AI vs AI or AI vs Human**

## Features

- **Player 1**: Human or Ollama AI (your choice)
- **Player 2**: Always Ollama AI
- **Simulate**: Watch AI vs AI battles automatically
- **Play**: Challenge the AI yourself

## Try It

**[Live Demo](https://html-preview.github.io/?url=https://github.com/Agent-Lumi/ai-game-simulator/blob/main/index.html)**

## Setup

1. Configure Ollama URL (default: `http://192.168.99.113:11434`)
2. Select AI model (llama3.2:3b, kimi-k2.5:cloud, mistral)
3. Choose Player 1: Human or AI
4. Click "Start Game" or "Auto-Play (AI vs AI)"

## How to Play

### Human vs AI
1. Select "👤 Human" for Player 1
2. Click "Start Game"
3. Click on the board to make your move
4. Watch AI respond!

### AI vs AI (Simulation)
1. Select "🤖 Ollama AI" for Player 1
2. Click "Auto-Play (AI vs AI)"
3. Watch the AIs battle it out!
4. Games auto-restart after each match

## AI Features

- Real-time decision making via Ollama API
- Strategic board analysis
- Fallback random moves if AI fails
- Game log tracking all moves

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Ollama API integration
- No dependencies, runs in browser
